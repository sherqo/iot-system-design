## System Functionality

The Smart Mosque Management System has 5 core functionalities:

### 1. Occupancy Detection
- PIR sensors at entrances count people entering/exiting
- Real-time occupancy count displayed on dashboard
- Triggers alerts when capacity exceeds safe limits (>80% = warning, 100% = critical)

### 2. Energy Management
- Auto turn ON/OFF lights and fans based on:
  - Prayer times (from API - aladhan.com)
  - Occupancy detection (motion sensors)
- Reduces unnecessary energy consumption

### 3. Water Monitoring (Wudu Area)
- Flow sensors measure water usage in real-time
- Detects abnormal patterns (leaks or taps left open)
- Solenoid valve automatically closes to prevent waste

### 4. Air Quality Monitoring
- IAQ sensors monitor CO₂, temperature, humidity
- Auto-activate ventilation when thresholds exceeded:
  - CO₂ > 1000 ppm
  - Temperature > 28°C
  - Humidity > 65%

### 5. Central Dashboard
- Displays: occupancy, energy, water, air quality
- Sends alerts for abnormal conditions
- Allows admin manual override control

### Devices Used in Packet Tracer

| Device | Purpose |
|--------|---------|

| PIR Motion Sensors | Occupancy detection |
| Flow Sensor (Water) | Water monitoring |
| IAQ Sensor | Air quality monitoring |
| Smart Relay (×2) | Control lights and fans |
| Solenoid Valve | Water flow control |
| IoT Server / Registration Server | Data accumulation |
| PC | Dashboard access |

### Connections

- Sensors → ESP32: Digital GPIO, I²C (for IAQ)
- ESP32 → Wi-Fi: IEEE 802.11 (WPA2-PSK)
- Wi-Fi → Server: Local LAN
- Server → PC: HTTP/Web browser
